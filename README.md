How to install the submod:

  To install the submod install the release on the "releases" tab. Make sure to download the "Submods.Zip" file, and not the release files!
  After you do that, simply extract the files into your Submods directory in Monika After Story.

How do I update the submod?
  Make sure you install the Submod updater plugin here; https://github.com/Booplicate/MAS-Submods-SubmodUpdaterPlugin
  If you have that installed, the submod will automatically tell you when there is an update, and will install it directly!
